VERCEL = Next.js UI ONLY. Python API = Render (editorial-engine-api.onrender.com).

=== REQUIRED (Dashboard) ===

1. Settings → General → Framework Preset: Next.js  (NOT Python)
2. Settings → General → Root Directory: frontend   (strongly recommended)
3. Settings → Environment → API_URL = https://editorial-engine-api.onrender.com
4. Deployments → Redeploy → Clear build cache

=== If you see "could not import app/main.py" ===

Vercel is deploying the FastAPI backend by mistake.
- Set Framework to Next.js
- Set Root Directory to frontend
- Repo .vercelignore excludes app/ and pyproject.toml

=== Test ===

  https://n-factorial.vercel.app/en          → 200 (Next.js)
  https://n-factorial.vercel.app/api/health  → 200 (proxy to Render)

Render API must be running with OPENAI_API_KEY set.
